# RobloxDrawVideosIdk
An automatic tool that turns your videos into frames in roblox drawing games then back to a video!
(i know the output frames looks bad so feel free to fix and modify the script)

# Do these
If you havent already installed these then run this:
```pip install opencv-python numpy pydirectinput keyboard scipy mss```

# How to use it
- Edit the configurations for your video in the script
- Run the python script and wait for the frames to load (remember to unequip your paint tool first)
- When its done loading, open roblox and watch it go!

(Be careful when using it though, because it controls your mouse during the proccess.)
